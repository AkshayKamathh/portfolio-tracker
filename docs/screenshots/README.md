# Screenshots (add your PNGs here)

These are not generated in CI—they are **your** captures after a local run with demo data loaded.

Suggested names (optional):

| File | What to show |
| ---- | ------------- |
| `01-overview.png` | Full page or top of app after **Load demo data** (header + summary). |
| `02-charts.png` | Charts section (value + benchmark + allocation). |
| `03-transactions.png` | Transactions table with filter or export visible. |
| `04-swagger.png` | FastAPI **/docs** with an expanded endpoint. |
| `05-pytest.png` | Terminal: `pytest` passing. |

**How to add:** save PNGs in this folder, `git add docs/screenshots/*.png`, commit, push.  
Then in the root `README.md` you can add an image line, for example:

```markdown
![Overview](docs/screenshots/01-overview.png)
```
